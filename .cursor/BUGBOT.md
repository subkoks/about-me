# Bugbot — about-me review rules

## Repository type

Markdown-only About page (activity graph + links). Profile README lives in `subkoks/subkoks`.

## Review focus

- Flag broken links to `subkoks/subkoks`, BSELA, or `agents-md` repos.
- Flag broken `github-readme-activity-graph` embed URL or invalid query parameters.
- Flag exposing private contact info, wallet addresses, or credentials.

## Non-blocking

- Copy edits, section reordering, or styling tweaks to match main profile README.
- Third-party graph CDN downtime — informational only.

## Scope

- No application code or tests required for README-only changes.
